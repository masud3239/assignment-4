
### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

Ans: difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll:

a. getElementById → Selects a single element by its unique id.

b. getElementsByClassName → Selects multiple elements by class name.

c. querySelector → Selects the first element that matches a CSS selector.

d. querySelectorAll → Selects all elements that match a CSS selector.
